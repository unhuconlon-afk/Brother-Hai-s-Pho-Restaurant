# Brother Hai's Pho Restaurant üçú

A modern, fully responsive restaurant website with interactive pho customization, real-time pricing, and order submission.

**Live Demo:** https://unhuconlon-afk.github.io/Brother-Hai-s-Pho-Restaurant/
**GitHub:** https://github.com/unhuconlon-afk/Brother-Hai-s-Pho-Restaurant

## Features
- Interactive pho builder (size, toppings, broth)
- Real-time price calculation with 5% online discount
- Order persistence using localStorage
- Form submission via Getform.org (no backend required)
- Mobile-first responsive design
- Clean, semantic HTML & well-structured CSS
- Deployed with GitHub Pages

## Tech Stack
- HTML5 / CSS3 (Flexbox, Grid, BEM)
- Vanilla JavaScript (ES6+)
- Getform.org for form handling
- Font Awesome + Google Fonts (Poppins)
